bme280::bme280 ( hwlib::i2c_bus &  i2c_bus,
uint8_t  address 
)

Constructor of bme280. Requires a i2c bus to use for communication with the bme280 and the i2c address, (usually 0x76 | 0x77) of the bme280. Default values that are set for the bme280 are normal mode and oversampling x2. This constructor does not perform any read or write operations with the bme280.

Parameters
[in]hwlib::i2c_bus&i2c_bus The hwlib i2c bus to use.
[in]uint8_tAddress The i2c address for the bme280.
